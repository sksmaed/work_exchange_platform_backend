# Host Module Requirements

## 1. Overview
The Host Module manages profiles of entities (businesses or individuals) providing work exchange opportunities, along with the actual vacancies/opportunities they offer.

## 2. Key Functional Requirements

### 2.1. Host Management
- Hosts must be linked to a core User.
- A host profile describes their business or location: Name, Type, Description, Address, Phone Number, and Contact Info.
- Hosts specify basic offerings: Pocket money, Meals offered, Dayoffs, Allowance, Facilities, and any other attributes.
- Hosts can express general preferences for arriving helpers: Expected duration, Licenses, Age, Gender, Personality, and Requirements.
- Support uploading a representative Host Image.
- Hosts possess a Recruitment Slogan.
- An Average Rating tracks public feedback on the host.

### 2.2. Vacancy Management
- Hosts can post multiple Vacancies (opportunities).
- Each Vacancy specifies its Name, Work Time, Description, and specific Expectations (Duration, Age, Gender, Licenses, Personality).
- Hosts can upload an image representing the Vacancy.
- Hosts can include Other Questions for the helpers applying.
- Track Vacancy status: Recruiting, Full, or Unavailable.
